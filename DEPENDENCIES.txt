PYTHON DEPENDENCIES (scripts/)
════════════════════════════════════════════════════════════════════════════════════

PRIMARY (Required for speech-to-text functionality):

1. openai-whisper (REQUIRED - installed during init)
   Purpose: OpenAI's speech-to-text transcription library
   Version: Latest (installs during initialization)
   License: MIT License
   License URL: https://github.com/openai/whisper/blob/main/LICENSE
   Repository: https://github.com/openai/whisper
   Install: pip3 install openai-whisper
   Note: Large models (base model ~140MB) are downloaded on first use

2. SpeechRecognition (RECOMMENDED)
   Purpose: Unified interface for multiple speech recognition engines
   Version: 3.x
   License: BSD 3-Clause License
   License URL: https://github.com/Uberi/speech_recognition/blob/master/LICENSE.txt
   Repository: https://github.com/Uberi/speech_recognition
   Install: pip3 install SpeechRecognition
   Supports:
     - Google Speech Recognition API (free, requires internet)
     - CMU PocketSphinx (offline, requires pocketsphinx package)
     - Other engines (Windows/Mac only)

OPTIONAL (For offline speech recognition):

3. pocketsphinx
   Purpose: Offline speech recognition engine (lightweight alternative to online APIs)
   License: BSD License
   License URL: https://github.com/cmusphinx/pocketsphinx/blob/master/LICENSE
   Repository: https://github.com/cmusphinx/pocketsphinx
   Install: pip3 install pocketsphinx
   Note: Lower accuracy than Google or Whisper, but works completely offline

4. PyAudio
   Purpose: Python bindings for audio I/O
   License: MIT License
   License URL: https://people.csail.mit.edu/hubert/pyaudio/ (MIT)
   Note: May require system-level audio libraries (portaudio)

STANDARD LIBRARY (Python):

5. sys, os, json, subprocess, pathlib
   Purpose: Core Python standard library modules
   License: Python Software Foundation License
   URL: https://docs.python.org/3/license.html

INITIALIZATION & INSTALLATION
════════════════════════════════════════════════════════════════════════════════════

When PMBuddy is run, it automatically handles Python dependency installation:

1. On first run: Check if openai-whisper is installed
2. If missing: Automatically install via pip3
3. Graceful fallback: If Whisper not available, use SpeechRecognition
4. If SpeechRecognition missing: Attempt to use system speech recognition

Installation commands (manual override):

  # Install all speech-to-text engines
  pip3 install openai-whisper SpeechRecognition pocketsphinx

  # Install minimal (Whisper only - recommended)
  pip3 install openai-whisper

  # Install minimal (Google API fallback)
  pip3 install SpeechRecognition
